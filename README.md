# Hi, I'm Alexis

I'm a **Vibe Coder and practical tool builder**. I use **Codex** as a coding partner to turn everyday problems into working browser tools, local automations and small data systems.

I like building at the intersection of **useful interfaces, clear data behaviour, automation and privacy-conscious workflows**. Most of my projects begin as a rough idea and evolve through real use, testing and iteration.

## What I'm working on

- **Codex-assisted building:** taking an idea from workflow sketch to working prototype, debugging and refinement
- **Local-first tools:** keeping personal records, backups and exports understandable and under the user's control
- **Learning tools:** browser apps for English, IELTS and PTE practice, diagnosis and progress tracking
- **Personal productivity:** schedule, timeline, notes, milestones and goal-management tools
- **Browser automation:** Chrome extensions for collecting, cleaning and exporting information from web pages
- **Small data systems:** transparent models, validation rules, CSV/JSON workflows and practical dashboards

## Projects built with Codex

- **Schedule Desk:** a private schedule workspace with timeline views, todos, notes, milestones, monthly goals, cloud sync and local backups
- **PTE Task-Type Weakness Estimator:** a transparent browser-based diagnostic model with confidence, risk and practice-priority guidance
- **Rednote Favorites Export:** a privacy-conscious Chrome extension that exports visible Xiaohongshu collection records to CSV or JSON
- **IELTS Reading PDF Helper:** a local workflow and Chrome extension for splitting reading pages into print-ready passage and question PDFs
- **English Typing Practice:** a lightweight vocabulary typing tool with custom word-list upload

## How I build

- Start with the real workflow, not a generic demo
- Keep the first version small but usable
- Make storage, privacy and failure behaviour explicit
- Use Codex to iterate quickly, then check the result through actual use
- Treat backup, export and recovery as part of the product
